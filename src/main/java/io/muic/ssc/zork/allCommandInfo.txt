*Available Commands*

[-starting game-]
- play {map-name}: play new game.
- load {saved-point-name}: load game state from saved point.
- save {saved-point-name}: save game state from saved point.

[-game play-]
- info: show information of the player and the current room.
- take: pick up the item.
- use: use the item that is not weapon.
- drop: drop the item.
- attack with: attack a monster.
- go {direction}: move to the room on that direction.
- map: show map.

[-other-]
- help: list all the commands.
- quit: go to home.
- exit: exit the game.
